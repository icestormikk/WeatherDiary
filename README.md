## WeatherDiary

Android-приложение, позволяющее пользователю вести собственный дневник погоды для своего города. Приложение помогает фиксировать погодные условия с конкретными параметрами и анализировать статистику по выбранным промежуткам времени.

## Функционал
- Добавление погодные записи за день с указанием:
  - даты и времени,
  - температуры (в °C),
  - атмосферного давления,
  - облачности,
  - погодных явлений (дождь, снег, гроза и др.),
  - ветра (скорость и направление).
- Просмотр списка всех записей погоды.
- Редактирование и удаление отдельных записей
- Фильтрация записей по:
  - диапазону дат
  - типу погодных явлений
  - уровню облачности
- Получение статистики за месяц, включая:
  - среднюю температуру
  - среднее давление
  - розу ветров (диаграмма распределения направлений ветра)
 
## Языки и инструменты
- Kotlin
- Android SDK
- SQLite (+ Room ORM)

1. Склонируйте репозиторий:

```bash
git clone https://github.com/icestormikk/WeatherDiary.git
```
2.Откройте проект
3. Подключите устройство или эмулятор Android
4. Соберите и запустите приложение на устройстве
